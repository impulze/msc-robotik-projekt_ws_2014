Projekt für LV Robotik (HS-RM, WS2014, MSc. Informatik)
=======================================================

In diesem Repository befinden sich sämtliche Dokumente und Quelltexte
für das Projekt der Wahl-Lehrveranstaltung.


Es folgt eine Beschreibung des Inhalts der CD, welche an den Dozenten
der Lehrveranstaltung bei der Projektvorstellung abgegeben wurde.

Das Verzeichnis 'code' enthält den Quelltext für das Programm, welches
im Projekt entwickelt wurde.
Darin befinden sich auch Beispielprojekte, so wie eine Datei für
ein Beispielraum.

Das Verzeichnis 'bin' enthält eine ausführbare Datei des Programms
für Ubuntu 14.04.1 und Windows 7.
Für Ubuntu 14.04.1 müssen die fehlenden Bibliotheken nachinstalliert werden.

Das Verzeichnis 'doc' enthält in 'praktikum.pdf' die Dokumentation
zum Projekt, sowie in 'präsentation.pdf' die Projektvorstellung als PDF-Dokumente.

In 'exercises' befinden sich die Quelltexte zu den Programmen aus den
Übungsaufgaben der Lehrveranstaltung.
